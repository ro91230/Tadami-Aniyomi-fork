# Инструкция по оформлению темы Tadami на 4PDA

Этот файл содержит готовый шаблон и рекомендации для создания темы вашего приложения на форуме 4PDA.

---

## Шаблон шапки темы (BBCode)

Скопируйте код ниже и вставьте его в редактор создания темы на 4PDA. Скриншоты уже настроены и подгрузятся автоматически с вашего GitHub репозитория.

```bbcode
[center][img]https://raw.githubusercontent.com/andreykolesnikov/Tadami-Aniyomi-fork/main/app/src/main/res/drawable-nodpi/ic_launcher_foreground.png[/img][/center]
[center][b][size=6][color=#7e57c2]Tadami[/color][/size][/b][/center]
[center][b]Версия: 0.19[/b][/center]

[b]Краткое описание:[/b]
Современный форк Aniyomi с фокусом на аниме, улучшенным UI (Aurora Theme) и удобным выбором озвучки.

[b]Описание:[/b]
Tadami — это модификация популярного приложения Aniyomi, созданная для тех, кто ценит эстетику и удобство. Приложение полностью переработано визуально с использованием темы [b]Aurora UI[/b], предлагая более чистый, современный и плавный интерфейс.

В отличие от оригинала, Tadami сфокусирован на просмотре аниме. Мы убрали лишние вкладки (манга), чтобы не перегружать интерфейс, и добавили ряд улучшений для комфортного просмотра.

[spoiler=Особенности и Фишки]
[list]
[*] [b]Aurora Theme:[/b] Полный визуальный редизайн. Элементы интерфейса стали приятнее, добавлены градиенты, улучшено выравнивание кнопок и карточек.
[*] [b]Умный Главный экран:[/b] Главный экран адаптируется под вас. После просмотра хотя бы одной серии, на главном экране автоматически появляется красивая информационная карточка с этим аниме для быстрого доступа.
[*] [b]Выбор озвучки:[/b] Продвинутый плеер (на базе MPV) позволяет легко переключать дорожки озвучки и субтитры в пару касаний прямо во время просмотра.
[*] [b]Поддержка AnimeGo:[/b] Полная совместимость с расширением для [i]animego.online[/i] (устанавливается отдельным APK), что дает доступ к огромной библиотеке тайтлов.
[*] [b]Фокус на Аниме:[/b] Упрощенное меню "Обзор". Вкладки, связанные с мангой, скрыты, чтобы вы могли сосредоточиться на поиске любимых тайтлов.
[*] [b]Синхронизация:[/b] Поддержка трекеров Shikimori, MyAnimeList, Anilist, Kitsu для отслеживания прогресса.
[/list]
[/spoiler]
```
